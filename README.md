## 如何生存 Markdown 文件目录

使用 [github-markdown-toc](https://github.com/ekalinin/github-markdown-toc) 插件。

### 安装

macOS (manual installation)

```shell
$ curl https://raw.githubusercontent.com/ekalinin/github-markdown-toc/master/gh-md-toc -o gh-md-toc
$ chmod a+x gh-md-toc
```

### 使用（例子）

```shell
./gh-md-toc ./HTML.md
```
